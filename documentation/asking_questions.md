# Asking Questions

## Questions about the Cell Painting Gallery

Please use the [issues page](https://github.com/broadinstitute/cellpainting-gallery/issues) of the Cell Painting Gallery GitHub repository to ask questions about the Cell Painting Gallery.

Questions in this category include questions about:

- the Cell Painting Gallery in general
- how to access data in the CPG
- file organization in the CPG
- artifacts missing from the CPG

## Questions about individual datasets

For questions about individual datasets that are hosted within the CPG, please ask in the repository associated with that dataset.
Dataset repository links can be found in our [Available Datasets](https://github.com/broadinstitute/cellpainting-gallery#available-datasets) documentation.

Questions in this category include questions about:

- biological interpretation
- methods and protocols used to generate the data
